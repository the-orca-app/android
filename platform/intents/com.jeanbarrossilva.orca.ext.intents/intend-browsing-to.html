intendBrowsingTo

inline fun intendBrowsingTo(uri: String, action: () -> Unit)

Asserts that an Intent for browsing to the given String representation of an Uri is started after the action lambda is run. The Intent responds with an "OK".

Parameters

uri

String version of the Uri to which browsing should be requested.

action

Callback that requests the browsing.

Throws

AssertionFailedError

If browsing to the uri is not requested or it is done so multiple times.
