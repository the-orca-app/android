StyledString

class StyledString(text: String, val styles: List<Style>) : CharSequence, Serializable

CharSequence that supports stylization such as emboldening, italicizing and linking (through plain URLs, e-mails, mentions and even simple text).

A StyledString can be created either through its constructors, or through buildStyledString.
