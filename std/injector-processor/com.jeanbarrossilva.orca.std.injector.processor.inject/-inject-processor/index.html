InjectProcessor

class InjectProcessor : SymbolProcessor

SymbolProcessor for ensuring the integrity of Inject-annotated properties and generating extension functions that retrieve the declared dependencies.

Reports an error if...

  • injections aren't part of a specific Module;

  • injections are private;

  • injections have a return type different from Module.() -> Any; or

  • a Module that declares injections is private.
