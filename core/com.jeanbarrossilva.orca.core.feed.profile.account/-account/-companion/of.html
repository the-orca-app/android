fun of(string: String, fallbackDomain: String? = null): Account

Parses the string into an Account.

It should be formatted as "{username}@{domain}", with "{username}" and "{domain}" replaced by the actual data they represent; it can also be given as "{username}", without the domain, if a fallbackDomain is specified.

Any leading or trailing whitespace in both the string and the fallbackDomain will be ignored.

Parameters

string

String to be parsed.

fallbackDomain

Domain to fallback to if the string only has a username.

Throws

Account.Companion.BlankStringException

If the string is blank.

Account.BlankUsernameException

If the username is blank.

Account.IllegalUsernameException

If the username contains any illegal characters.

Domain.BlankValueException

If the domain is not specified in the string and the fallbackDomain is null.

Domain.IllegalValueException

If the domain contains any of the Domain.illegalCharacters.

Domain.ValueWithoutTopLevelDomainException

If the domain doesn't have a top-level domain.
