fun of(string: String): Follow

Parses the string into a Follow.

string must be the result of calling toString on one of the existing Follows (e. g., Follow.of("${Follow.Public.unfollowed()}") returns Public.unfollowed). Any leading or trailing whitespace it may contain will be ignored.

Return

Follow whose result of toString equals to the string (minus surrounding whitespaces).

Parameters

string

String to be parsed into a Follow.

Throws

Follow.Companion.BlankStringException

If the string is blank.

Follow.Companion.InvalidFollowString

If the string is not a valid Follow representation.
