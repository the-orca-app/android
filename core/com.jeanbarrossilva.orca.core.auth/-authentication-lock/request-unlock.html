requestUnlock

suspend fun <T> requestUnlock(listener: AuthenticationLock.OnUnlockListener<T>): T

Ensures that the operation in the listener's onUnlock callback is only performed if the Actor is authenticated; if it isn't, then authentication is requested and, if it succeeds, the operation is performed.

Return

Result of the listener's onUnlock.

Parameters

T

Value returned by the listener's onUnlock.

listener

OnUnlockListener to be notified if the Actor is authenticated.

Throws

AuthenticationLock.FailedAuthenticationException

If authentication fails.
