ContextualMastodonInstance

class ContextualMastodonInstance(context: Context, domain: Domain, authorizer: MastodonAuthorizer, val authenticator: MastodonAuthenticator, actorProvider: ActorProvider, val authenticationLock: AuthenticationLock<MastodonAuthenticator>, termMuter: TermMuter, imageLoaderProvider: ImageLoader.Provider<URL>) : MastodonInstance<MastodonAuthorizer, MastodonAuthenticator>

MastodonInstance that authorizes the user and caches all fetched structures through the given Context.

Parameters

context

Context with which the authenticator will be created.

domain

Unique identifier of the server.

authorizer

MastodonAuthorizer by which the user will be authorized.

actorProvider

ActorProvider that will provide Actors to the authenticator, the authenticationLock and the feedProvider.

termMuter

TermMuter by which Posts with muted terms will be filtered out.

imageLoaderProvider

ImageLoader.Provider that provides the ImageLoader by which Images will be loaded.
