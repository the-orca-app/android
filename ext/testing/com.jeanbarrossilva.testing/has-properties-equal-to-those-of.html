hasPropertiesEqualToThoseOf

inline fun <F : Any, S : Any> Assert<F>.hasPropertiesEqualToThoseOf(other: S): Assert<F>

Asserts that the value's properties have the same name and hold data equal to other's. It will not fail if it has ones that aren't present in other, vice-versa or none at all, only if the ones that are named equally have been assigned differently.

Parameters

F

Value on which the assertion will be performed.

S

Value to be compared to F.

other

Object whose properties will be compared to the value's.

Throws

AssertionFailedError

If a property of the value has a counterpart in other but has different data assigned to it.
