Notifier

class Notifier

No-op structure that serves the sole purpose of being emitted to a notifierFlow, allowing it to send notifications for other s that collect it to emit new values.

A call to next returns either the initialNotifier if this is the subsequent one or the subsequent if this is the initial one, allowing for them to be emitted interchangeably, working around the constraint of StateFlows that prevents them from emitting equal values consecutively.
